# Это директория с веб апишкой для инференса модели

## Для ML-модели (SVC над фичами HOG)

Для запуска введите в терминале:

1. `uvicorn app:app --reload`
2. После чего перейдите по адресу `http://127.0.0.1:8000/docs`
3. В методе `POST` выбираете `try it out` и загружаете свое изображение
4. В response получим метку класса наиболее похожего кроссовка — profit!

Перед этим убедитесь, что все скачано из **requirements.txt**.

Тут реализован метод **POST**, который по загруженному изображению запускает модель и возвращает предсказание класса кроссовок, которые есть на изображении.

Возвращает данные в формате

```python
    {
        "filename": image.filename,
        "predicted_class": predicted_class
    }
```

## Для ResNet18

Всё аналогично, только в приложение надо запускать как `uvicorn resnet_app:app --reload`
