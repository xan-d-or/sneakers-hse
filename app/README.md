Для запуска приложения из корня проекта надо выполнить

```bash
pip3 install -r requirements.txt
pip3 install -e .
python3 -m uvicorn app.main:app --reload
```

После чего перейдите по адресу `http://127.0.0.1:8000/docs`

В методе `POST` выбираете `try it out` и загружаете свое изображение

В response получим ссылки на похожие кроссовки среди нашей обучающей выборки.

Тут реализован метод **POST**, который по загруженному изображению запускает модель и возвращает предсказание класса кроссовок, которые есть на изображении.
